ThemeDock

Self-hosted custom CSS themes for Unraid and your containers. Grab a URL, snippet, or reverse-proxy config below.

Unraid

Standalone, self-contained stylesheets. Install the Custom WebUi CSS plugin from Community Applications, then paste the raw URL into its settings field.

Multi-app themes

Two-layer theming: each app has a base stylesheet, and a theme-options file sets the palette on top of it. None of these apps have a native custom CSS field, so apply them at your reverse proxy instead. "Copy NPM config" gives you a ready-to-paste Nginx Proxy Manager snippet (Advanced tab → Custom Nginx Configuration); "Copy @import snippet" is for setups that do accept raw CSS text (e.g. Stylus).
