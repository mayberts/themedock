ThemeDock

Self-hosted custom CSS themes for Unraid and your containers — a theme-park-style catalog you point your apps' custom CSS at.

Unraid

Standalone, self-contained stylesheets — one raw URL, paste it into Unraid's custom CSS field.

theme.park-style apps

Two-layer theming: each app has a base stylesheet, and a theme-options file sets the palette on top of it. Copy the snippet for your app and paste both lines into that app's custom CSS field.

Built with themes/manifest.json · contribute a theme via pull request on GitHub · base CSS from theme.park (MIT)
