ThemeDock

Self-hosted CSS themes for Unraid and your containers.

Unraid

Install the Custom WebUi CSS plugin from Community Applications, then paste the raw URL into its settings field.

Multi-app themes

None of these apps have a native custom CSS field, so themes apply at your reverse proxy instead. Pick an app, then "Copy NPM config" for Nginx Proxy Manager (Advanced tab) or "Copy @import snippet" for setups that accept raw CSS text.

Built with themes/manifest.json · contribute a theme via pull request on GitHub
